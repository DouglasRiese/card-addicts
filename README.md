# CWEB280_Project Card Addicts

## Setup Instructions

Install npm packages in package.json. Run in root directory.

```bash
npm i
```

## Running Instructions

1. Start server with

```bash
node ./bin/www
```

2. Go to on [http://localhost:3001](http://localhost:3001)

Note: (Uses port `3001`)

## Coding Tips for Project

To use an image path in a JS file just put `/images/image.png` this will allow you to use any image in
the `/public/images/` folder.
